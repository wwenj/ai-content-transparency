# AI Content Transparency

[中文](./README.md)

`ai-content-transparency` is a lightweight open project for transparent AI participation disclosure in content creation. It provides a simple declaration spec and a browser-based card generator.

## Why This Project

AI-assisted writing is already common, but disclosure is often vague or inconsistent. This project proposes a practical, voluntary way to make AI participation visible.

## How to Use

1. Read [SPEC.en.md](./SPEC.en.md) and decide `ai_ratio`, `human_ratio`, `ai_roles`, and `human_roles`.
2. Place the declaration at the beginning of the article or page.
3. Open [generator/index.html](./generator/index.html) in a browser, generate a card, and export it as PNG if needed.

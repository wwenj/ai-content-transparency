# AI Content Transparency

[English](./README.en.md)

`ai-content-transparency` 是一个用于内容创作中 AI 参与透明声明的轻量开源项目，提供一份简洁规范和一个浏览器卡片生成工具。

## 为什么需要这个项目

AI 辅助创作已经很常见，但声明方式经常模糊、不统一。这个项目希望提供一种简洁、实用、以自愿为前提的透明表达方式。

## 如何使用

1. 阅读 [SPEC.md](./SPEC.md)，确定 `ai_ratio`、`human_ratio`、`ai_roles`、`human_roles`。
2. 将声明放在文章或页面开头。
3. 用浏览器打开 [generator/index.html](./generator/index.html)，生成卡片并按需导出 PNG。

# AI Content Transparency

[English](#english) | [中文](#中文)

## English

`ai-content-transparency` is a lightweight open project for transparent AI participation disclosure in content creation. It includes a simple declaration spec and a browser-based card generator.

Why this project: AI-assisted writing is now common, while disclosure is often inconsistent or vague. This project proposes a practical, voluntary, non-punitive way to make AI participation visible.

How to use:
1. Read [SPEC.md](./SPEC.md) and decide your `ai_ratio`, `human_ratio`, `ai_roles`, and `human_roles`.
2. Put the declaration at the beginning of the article or page.
3. Open [generator/index.html](./generator/index.html) in a browser, generate a card, and export it as PNG if needed.

Example:
- Usage example: [examples/usage.md](./examples/usage.md)
- Suggested sample image path: `examples/simple.png`
- Full manifesto: [MANIFESTO.md](./MANIFESTO.md)

## 中文

`ai-content-transparency` 是一个用于内容创作中 AI 参与透明声明的轻量开源项目，提供一份简单声明规范和一个浏览器卡片生成工具。

为什么需要它：AI 辅助创作已很常见，但声明方式经常模糊、不统一。这个项目希望提供一种实用、非惩罚、以自愿为前提的透明表达方式。

如何使用：
1. 阅读 [SPEC.md](./SPEC.md)，确定你的 `ai_ratio`、`human_ratio`、`ai_roles`、`human_roles`。
2. 将声明放在文章或页面开头。
3. 用浏览器打开 [generator/index.html](./generator/index.html)，生成卡片并按需导出 PNG。

示例说明：
- 使用示例见 [examples/usage.md](./examples/usage.md)
- 示例卡片建议放在 `examples/simple.png`
- 公约全文见 [MANIFESTO.md](./MANIFESTO.md)

# Usage Example

Below is a complete example of how to place the declaration at the beginning of an article.

## Markdown Example

```md
> AI Content Transparency / AI 内容透明声明
> AI Ratio / AI 占比: 35%
> Human Ratio / 人类占比: 65%
> AI Roles / AI 分工: 提纲起草、语言润色、标题建议
> Human Roles / 人类分工: 选题确定、论证设计、事实复核、最终审核
> Note / 说明: 本声明为作者自愿披露，用于说明 AI 在本文创作中的参与情况。

# 为什么内容创作需要 AI 透明声明

正文从这里开始……
```

## Short HTML Example

```html
<section class="ai-disclosure">
  <strong>AI Content Transparency / AI 内容透明声明</strong><br>
  AI Ratio / AI 占比: 35%<br>
  Human Ratio / 人类占比: 65%<br>
  AI Roles / AI 分工: 提纲起草、语言润色、标题建议<br>
  Human Roles / 人类分工: 选题确定、论证设计、事实复核、最终审核
</section>
```

## Notes

- Put the declaration before the main body.
- Keep the wording specific.
- Do not use the declaration as a marketing slogan or misleading trust badge.

# 使用示例

[English](./usage.en.md)

下面给出一个完整示例，说明如何在文章开头放置声明。

## Markdown 示例

```md
> AI 内容透明声明
> AI 占比：35%
> 人类占比：65%
> AI 分工：提纲起草、语言润色、标题建议
> 人类分工：选题确定、论证设计、事实复核、最终审核
> 说明：本声明为作者自愿披露，用于说明 AI 在本文创作中的参与情况。

# 为什么内容创作需要 AI 透明声明

正文从这里开始……
```

## HTML 示例

```html
<section class="ai-disclosure">
  <strong>AI 内容透明声明</strong><br>
  AI 占比：35%<br>
  人类占比：65%<br>
  AI 分工：提纲起草、语言润色、标题建议<br>
  人类分工：选题确定、论证设计、事实复核、最终审核
</section>
```

## 使用提示

- 声明放在正文开始之前
- 分工描述尽量具体
- 不要把声明当作营销口号或误导性背书
